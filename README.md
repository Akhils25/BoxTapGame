BoxTapGame is a simple and fun Android game built using **Jetpack Compose**.  
The goal is to tap the animated box as quickly and accurately as possible to score points.

This project is designed to demonstrate:
- Modern Android UI with Jetpack Compose
- Simple game logic
- Animations using Compose
- Splash screen with animated Box
- Clean and beginner-friendly architecture

---

## Features

- Splash screen with animated Box
- Tap-the-box gameplay
- Score tracking
- Simple game loop
- Material 3 UI
- Fully built with Jetpack Compose (No XML)

---

## Tech Stack

- **Language:** Kotlin
- **UI:** Jetpack Compose
- **Architecture:** Composable-based UI
- **Design:** Material 3
- **Minimum SDK:** 26+

---

## Screens (Planned)

- Splash Screen (Animated Box)
- Game Screen
- Score Display

---

## Game Rules

1. A box appears on the screen
2. Tap the box to increase your score
3. Faster taps = higher score
4. Simple, fun, and addictive

## Learning Purpose

This app is ideal for:
- Beginners learning Jetpack Compose
- Understanding state management
- Working with animations
- Building simple games in Compose
